Introducción

El objetivo de este trabajo es aplicar las técnicas aprendidas en la materia, aplicando las etiquetas necesarias para crear un sitio web funcional, así como también aplicar los conocimientos obtenidos en el diseño gráfico para una página web, como la paleta de colores, el maquetado, el tipo de fuente, etc.

La siguiente página cumple con los objetivos presentados en la lista de ejercicios, ya que utilizamos enlaces internos, externos y locales, así como también diversas imágenes para ilustrar la distintas páginas creadas

Este sitio web cuenta con 4 pestañas adicionales donde se habla de los temas proporcionados por la profesora, para poder incorporarlos todos decidimos crear un sitio web de recomendaciones para distintas formas de entretenimiento, en donde el usuario puede hacer click en el medio de preferencia y los llevará hacia la pestaña que contiene una recomendación para su gusto.
